# 律 (Lyu) 系列静态站点

一个用于展示「律」系列应用的静态网页，包含已上线与即将上线的 App 卡片展示。

## 预览
- 打开 `index.html` 即可在浏览器查看。

## 目录结构
- `index.html`：页面结构与数据渲染（JS 数据驱动卡片）。
- `style.css`：页面样式与背景方案。
- `images/`：应用图标。
- `data/apps.json`：应用与多平台配置（含上线/待上线状态）。

## 背景方案说明
`style.css` 中提供多种背景方案（当前默认微妙网格纹理），可按注释切换。

## 开发/部署提示
- 静态站点，可直接放置于 GitHub Pages（例如 `lyu-series.github.io`）。
- 如使用构建工具，常见忽略项已在 `.gitignore` 中列出。
